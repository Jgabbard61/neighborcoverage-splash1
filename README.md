# NeighborCoverage - Auto Insurance Splash Page

A high-converting lead generation splash page for auto insurance with:
- 5 prominent click-to-call CTAs
- Google Analytics 4 tracking
- Mobile-optimized sticky call button
- Professional branding and design

## Features
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ GA4 event tracking for all CTAs
- ✅ Click-to-call functionality
- ✅ Animated elements and hover effects
- ✅ SEO-optimized metadata

## Tech Stack
- Next.js 14.2.28
- TypeScript
- Tailwind CSS
- Lucide React Icons

## Deployment

This site is optimized for Vercel deployment.

### Deploy to Vercel
1. Push this repo to GitHub
2. Import project in Vercel
3. Deploy (zero configuration needed)
4. Configure custom domain: `neighborcoverage.com`

### Environment Variables
None required! This is a static site.
